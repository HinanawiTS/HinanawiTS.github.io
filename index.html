My Projects

▶ Wildfire Hazard Analysis in San Diego County

In this project, I conducted a suitability analysis in San Diego, using ArcGIS Python API I was able to use landcover, landsat (satellite imagery) and rainfall information to assess the "suitability" of wildfires. This was a solo project. Access the report.

▶ Indoor Airborne Infection Risk Estimator

An online COVID – 19 infection risk estimator which features interactive visualization, great transparency and extensibility. I built the entire frontend of the website, and the majority of its backend which is a dash-based Python application. Visit the website.

▶ Political Tendency Analysis and U.S Election Prediction

In this project, we analyzed how income, population and geospatial features of counties in the U.S correlated to their political affiliation, and used features derived from them to predict election results in those counties. The analysis revealed certain geospatial features were closely related to local political affiliations. The resulting model achieved an accuracy of over 90% for predicting 2016 election using 2012 dataset. I built the majority of this project. Access the report.
